## Creating Your Profile

### Required Information

When joining WebChat, you'll need to provide:

1. **Username** (3-20 characters)
   - Must be unique
   - Choose carefully - this is how others will see you

2. **Age** (13-120 years)
   - Must be at least 13 years old
   - Displayed next to your username

3. **Sex** (Female or Male)
   - Determines your username color:
     - **Pink** for Female users
     - **Blue** for Male users

4. **Country**
   - Select from dropdown list
   - Helps connect with users from your region

### Registration Options

After entering your information, you have two choices:

#### Option 1: Anonymous User (No Registration)
- Skip the password field
- Join immediately
- **Note**: Your username can be taken by someone else later
- You cannot recover your account if you log out

#### Option 2: Registered User (Recommended)
- Set a password (minimum 6 characters)
- Your username is **reserved** for you
- Log back in anytime with your credentials
- Secure your identity

### Login for Returning Users

If your username is already registered:
1. Enter your username and profile information
2. You'll be prompted to enter your password
3. Click "Login" to access your account

If you forgot your password:
- Choose a different username to start fresh
- Contact an administrator for assistance

## Chat Rooms

### Viewing Rooms

- Room list appears on the **left sidebar**
- Shows all available chat rooms
- Click any room to join

### Joining a Room

1. Click on a room name in the left sidebar
2. The room will load with recent messages
3. You'll see other users currently in the room

### Sending Messages

1. Type your message in the input box at the bottom
2. Press **Enter** or click **Send**
3. Messages appear instantly for all users in the room
4. Maximum message length: **5000 characters**

### Creating a Room

1. Click the **"+ Create Room"** button in the room list
2. Enter a unique room name
3. The room will be created and you'll join automatically

### Leaving a Room

- Simply select a different room
- Or close the browser tab

## Direct Messages

### What are Direct Messages?

Direct Messages (DMs) allow you to chat privately with any online user.

### Starting a DM Conversation

1. Look at the **Online Users** sidebar on the right
2. Click on any user's name
3. A private chat window opens
4. Type and send messages (max 1000 characters)

### Features

- **Real-time delivery** - Messages appear instantly
- **Private** - Only you and the recipient see the messages
- **History** - Conversation history is saved
- **Timestamps** - Each message shows when it was sent

### Message Appearance

- **Your messages**: Blue bubbles on the right
- **Their messages**: Gray bubbles on the left

### Switching Between Rooms and DMs

- Click **"Close"** in the DM window to return to room chat
- Click a room name to switch to room chat
- Click a user name to switch to their DM

## Online Users

### Online Users Sidebar

Located on the **right side** of the screen, showing:

- All currently connected users
- Format: **Username (age)**
- Color coding by sex:
  - **Pink** for Female
  - **Blue** for Male

### User Count

- Total number of online users displayed at the top
- Updates in real-time as users connect/disconnect
- You are **not** shown in your own list

### Interacting with Users

- **Click** on a user to start a DM conversation
- **Block/Unblock** button next to each user
- Blocked users appear grayed out

## Blocking Users

### Why Block a User?

Block users who:
- Send unwanted messages
- Harass or spam you
- You don't want to communicate with

### How to Block

1. Find the user in the **Online Users** sidebar
2. Click the **"Block"** button next to their name
3. They are immediately blocked

### What Happens When You Block Someone?

- They **cannot** send you direct messages
- Their messages are rejected by the system
- They appear **grayed out** in your online users list
- They can still see you online (they're not notified of the block)

### How to Unblock

1. Find the blocked user (they'll have "Blocked" label)
2. Click the **"Unblock"** button
3. They can now send you DMs again

### Important Notes

- Blocking is **one-way** - You can still send them messages
- They won't receive a notification that you blocked them
- You can block/unblock as many times as needed

## Account Types

### Anonymous Users

**Advantages:**
- Quick access - no password needed
- Temporary usage - great for one-time visits
- No commitment

**Disadvantages:**
- Username can be taken by others
- Cannot recover account after logout
- No persistent identity

### Registered Users

**Advantages:**
- **Reserved username** - nobody else can use it
- Login from any device
- Persistent account
- Account recovery possible

**Disadvantages:**
- Must remember password
- Slightly longer signup process

**Recommendation:** Register if you plan to use WebChat regularly!

## Privacy & Security

### Your Privacy

- **Usernames are public** - visible to all users
- **Age is public** - shown next to your username
- **Sex is public** - determines your color coding
- **Country is public** - shown in your profile
- **Messages are stored** - both room and direct messages

### Security Best Practices

1. **Choose a strong password** (if registering)
   - At least 6 characters
   - Mix letters and numbers
   - Don't share with anyone

2. **Don't share personal information**
   - No real names (unless you want to)
   - No addresses or phone numbers
   - No sensitive financial information

3. **Block troublesome users immediately**
   - Don't engage with harassment
   - Use the block feature
   - Report serious issues to admins

4. **Log out when finished**
   - Especially on shared computers
   - Protects your account

### Message Security

- Messages are sanitized to prevent malicious code
- XSS protection is enabled
- All messages stored in secure database

### Data Collection

WebChat collects:
- Username, age, sex, country (for profile)
- Message content (for chat history)
- Connection timestamps
- Blocked users list (for your privacy)

WebChat **does not** collect:
- Email addresses
- IP addresses (beyond session management)
- Browsing history
- Personal identifying information

## Troubleshooting

### Cannot Login

**Problem:** "Password required for this username"
- **Solution:** This username is registered. Enter the correct password or choose a different username.

**Problem:** "Username already exists"
- **Solution:** This username is taken by an anonymous user. Choose a different username.

### Messages Not Sending

**Problem:** Message doesn't appear
- **Check:** Are you connected? Look for "disconnected" message
- **Check:** Is your message within character limit?
- **Solution:** Refresh the page and try again

**Problem:** DM not delivering
- **Check:** Is the user still online?
- **Check:** Did they block you?
- **Solution:** Try sending to a different user

### Cannot See Online Users

**Problem:** Online users list is empty
- **Check:** Are other users actually online?
- **Solution:** Wait for other users to connect
- **Solution:** Refresh the page

### Cannot Block/Unblock User

**Problem:** Block button doesn't work
- **Check:** Are you clicking on yourself? (Can't block yourself)
- **Solution:** Refresh the page and try again

## Frequently Asked Questions

### Can I change my username?

- **Anonymous users**: Yes, logout and create a new profile
- **Registered users**: No, username is permanent. Create a new account if needed.

### Can I change my age/sex/country?

Currently, no. You must create a new account to change profile information.

### What happens to my messages when I log out?

- **Room messages**: Saved permanently, visible to all users
- **Direct messages**: Saved permanently, visible to you and recipient
- **Account**: Anonymous accounts have no persistence; registered accounts persist

### Can blocked users see that I blocked them?

No, blocking is invisible. They simply cannot send you DMs.

### How long do messages stay in the system?

Messages are stored indefinitely. Admins may periodically clean old data.

### Is WebChat mobile-friendly?

Yes! WebChat works on phones, tablets, and computers.

### Can I use emojis?

Yes! Type emojis using your device's emoji keyboard. They'll display for all users.
